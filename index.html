Search, filter, and live-query your web service API documentation.

Features

Apiculate is an HTML5 API documentation viewer that allows you to write up your endpoints in JSON, search them, filter them, and send live requests to your server.

  • Describe your web service endpoints as JSON
  • Search through endpoint's url and description
  • Filter on HTTP verb and flags like 'Requires authentication'
  • Live query servers with parameters you supply

Setup

Configuring apiculate is easy: just drop in some JSON in a script in the <head> element.

			
			<head>
				<!-- your script goes here-->
				<script>		
					window.apiculate.config = {
						//let's fill this with JSON!
					};
				</script>
				
			</head>
